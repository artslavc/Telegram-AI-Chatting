## О Программе
Небольшая "Теория мертвого интернета" в реальности.
К скрипту подключается аккаунт, который активно беседует в группах телеграмма.
Используется ChatGPT-4.
В ходе эксперимента было подключено 2 аккаунта, которые в одной из публичных групп вели активную беседу. При этом иногда отвечая друг другу и ведя осмысленный диалог.

## Конфигурация
groups.txt - Здесь перчислены группы, в которых скрипт должен вести беседу. (1 строчка = 1 группа)
prompt.txt - Промпт для ChatGPT.

## Установка зависимостей
